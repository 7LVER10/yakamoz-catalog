## 📝 Как активировать HERMES

### Вариант 1 — Полный запуск
1. Открой любой LLM (ChatGPT, Claude, Gemini, Groq)
2. Вставь весь содержимое HERMES.md как System Prompt
3. Добавь содержимое HERMES-RULES.md
4. HERMES активирован

### Вариант 2 — Быстрый запуск
Скопируй блок выше и отправь как System Prompt напрямую в LLM.

### Вариант 3 — N8N Integration
Используй содержимое как `systemMessage` в N8N AI Agent ноде.
