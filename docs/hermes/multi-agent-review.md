# Multi-Agent Review (LLM Council Method)

Метод повышения точности советов через проверку несколькими LLM/агентами.

## Суть метода

Один LLM склонен говорить то, что хочется услышать. Метод "LLM Council" (Карпати) решает это: несколько независимых агентов анонимно проверяют ответ друг друга и выдают общий вердикт.

## Как применять в нашей системе

1. **Когда подключать:** при принятии важных решений (выбор архитектуры, смена модели, критические советы)
2. **Какие агенты участвуют:** Hermes (локальный), MiMo (облачный), при необходимости — внешний LLM (Gemini/Claude)
3. **Процесс:** каждый агент получает одинаковый вопрос независимо → сравниваются ответы → итоговый вердикт = консенсус или явное расхождение
4. **Фиксация вердикта:** один итоговый ответ с пометкой "council review" и ссылкой на участников
5. **Ограничение:** не для каждого запроса, только для решений с высокой ценой ошибки
