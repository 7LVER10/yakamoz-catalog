## Quick Install

### UI UX Pro Max (Claude Code)
```bash
npm install -g uipro-cli
uipro init --ai claude --global
```

### Magic MCP (Cursor / Windsurf)
```bash
npx @21st-dev/cli@latest install --api-key YOUR_API_KEY
```
Get API key: https://21st.dev/magic/console
